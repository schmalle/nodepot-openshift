Openshift
---------

Openshift is a great cloud solution from Red Hat. It offers free hosting for NodeJS.

1. Create an account at openshift

2. Create your own application

rhc app create <your app name> nodejs-0.10 --from-code=http://github.com/schmalle/nodepot.git

3. Add redis to your application

rhc add-cartridge http://cartreflect-claytondev.rhcloud.com/reflect?github=smarterclayton/openshift-redis-cart --app <your app name>

4. ssh into your application

5. copy files around as needed

/bin/bash ./app-root/repo/openshift.sh




(now copy all stuff)


    "start": "cd $HOME;/bin/bash ./app-root/repo/openshift.sh;nodejs app.js"





 rhc cartridge-status redis -a myapp

I then set ENV variables like so:

rhc set-env OPENSHIFT_REDIS_HOST=xxhostnumberxx-myapp.rhcloud.com -a myapp
rhc set-env OPENSHIFT_REDIS_PORT=com:xxportnumberxx -a myapp
rhc set-env REDIS_PASSWORD=password:xxsomepasswordxx -a myapp
